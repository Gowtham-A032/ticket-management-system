# 🎟️ Ticket Management System (C Project)

A console-based ticket management system simulating bus transport ticketing operations.  
Developed in **C language**, this project demonstrates structured programming, modular design, and file handling concepts.

---

## 📖 Description
This project, **Inter-City Transport Corporation (IRTC)**, is a prototype ticketing system for government bus services.  
It allows users to generate tickets, passes, and free tickets under welfare schemes, while dynamically calculating fares and storing records in text files.

**Features:**
- Regular tickets: Senior Citizen, Half Ticket, Full Ticket
- Travel passes: Daily, Weekly, Monthly
- Free tickets: Women Empowerment Scheme
- Dynamic fare calculation based on routes
- File handling for ticket storage (`print.txt`)
- Ticket counter to track daily sales

---

## 🛠️ Methodology
The system is divided into modules:
1. **Ticket Generation** – Accepts passenger type & payment mode, calculates fare, applies discounts, prints ticket.
2. **Pass Generation** – Generates day/weekly/monthly passes with ID verification.
3. **Free Ticket Generation** – Issues zero-fare tickets for women with Aadhaar verification.
4. **Display Locations** – Lists 9+ travel destinations.
5. **File Handling** – Saves ticket/pass data into `print.txt`.
6. **Ticket Counter** – Displays total tickets sold for the day.

**Programming Concepts Used:**
- Loops & arrays
- Switch-case structures
- Functions & modular design
- File handling (`fopen`, `fprintf`, `fclose`)
